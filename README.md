# Breakout-game


This is a sample "Break Out" style game, it is made using Corona SDK.

To run this game follow the below steps:

1) Install Corona SDK from https://coronalabs.com/products/corona-sdk/
2) Launch Corona SDK and run the main.lua
3) Game will be launched in Corona Simulator

Controls: tap on the right part of the screen to move the paddle right and tap on the left part of the screen to move paddle left
